# Alcohol Consumption Analysis

This repository contains an exploratory data analysis of global alcohol consumption patterns across 193 countries. The analysis evaluates beverage consumption trends (beer, spirits, and wine) and total litres of pure alcohol consumed per capita.
Using the dataset, the project explores how alcohol consumption differs across countries and identifies global patterns in drinking preferences. The analysis was conducted using Python and the Pandas library for data exploration and statistical summarization.
The dataset includes key alcohol consumption indicators such as beer servings, spirit servings, wine servings, and total litres of pure alcohol consumed.

## Highlights

- Dataset contains **193 countries and 5 variables** describing alcohol consumption behavior.
- Beer consumption varies significantly across countries, with some countries consuming **over 300 servings annually**.
- Spirit consumption shows strong variation between regions.
- Wine consumption is highly concentrated in specific countries.
- Total litres of pure alcohol provides a unified measure for comparing overall alcohol consumption.

## Summary Findings

- **Global Consumption Patterns**: Average alcohol consumption across countries is approximately **4.7 litres of pure alcohol per capita**. However, the distribution is highly uneven, with several countries reporting near zero consumption while others exceed 12 litres.
- **Beverage Preferences**: Alcohol consumption preferences vary significantly: **Beer** tends to dominate in many countries with high serving averages. **Spirits** are popular in certain regions where distilled alcohol is culturally common. **Wine** consumption is concentrated in specific countries known for wine production and culture.
- **Variation Across Countries**: Some countries report **very high alcohol consumption**, while others show extremely low levels due to cultural, religious, or regulatory factors.
- **Data Distribution**: The dataset shows high variance across all serving categories, indicating significant global diversity in drinking habits.
