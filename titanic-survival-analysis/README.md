# Titanic Survival Analysis

This project explores the Titanic dataset to understand factors that influenced passenger survival.

## Objectives
- Explore the structure and quality of the dataset
- Perform descriptive and exploratory data analysis
- Identify relationships between passenger characteristics and survival

## Key Analysis
- Survival distribution by gender and passenger class
- Age and fare analysis
- Visualization of survival patterns

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
